51 Never Do

Visual

  • Skeleton loaders (shimmer only)
  • Light mode / white backgrounds in app UI (ONLY exception: PDF report export)
  • Sidebar navigation (topbar only)
  • Fonts beyond Space Grotesk + Manrope + DM Mono
  • Box shadows on cards (borders + gradient-border only)
  • Emoji icons (Lucide line only)
  • Font size below 9px
  • Instant card appearance (always fade + stagger)
  • Hardcoded hex in JSX (use CSS vars / Tailwind tokens)
  • Tailwind emerald/amber/red for scores (use AVQ tokens)

Interaction

  • Save buttons on settings (live-edit + 3s debounce)
  • Plus/minus for numbers (use sliders)
  • Checkboxes f
